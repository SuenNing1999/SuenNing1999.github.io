《数据结构》C语言版——单链表学习笔记

发表于 2023-02-01 阅读次数：
本文字数： 8k 阅读时长 ≈ 7 分钟

链表中的数据是以结点来表示的，每个结点的构成：**元素(数据元素的映象) + 指针(指示后继元素存储位置)**，元素就是存储数据的存储单元，指针就是连接每个结点的地址数据。

●头结点是在链表的开始结点之前附加的一个结点(结点没有有效的数据元素)
●第一个结点（或称首元结点）是链表中存储第一个有效数据元素的结点
